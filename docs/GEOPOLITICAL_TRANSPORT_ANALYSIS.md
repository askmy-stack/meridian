### 1.2 Pattern Recognition: The "Chokepoint Cascade" Effect

Historical analysis reveals a predictable cascade pattern when conflicts affect strategic chokepoints:

```
Phase 1 (0-2 weeks): Immediate closure/denial
  → Alternative route activation
  → 15-40% cost increase
  → Insurance premium spikes

Phase 2 (2-12 weeks): Rerouting stabilization
  → New equilibrium pricing
  → Capacity constraints emerge
  → Modal shift (sea→air, sea→rail)

Phase 3 (3-6 months): Adaptive restructuring
  → Long-term contract renegotiations
  → New trade corridors established
  → Infrastructure investments in alternatives

Phase 4 (6+ months): Permanent landscape change
  → Diversification becomes strategic imperative
  → Regional realignment of trade flows
  → Military presence in shipping lanes increases
```

```python
# Simplified predictive model based on historical conflict data
def forecast_shipping_costs(conflict_intensity, chokepoint_status, time_horizon_months):
    """
    Parameters:
    - conflict_intensity: 0-10 scale (0=peace, 10=full war)
    - chokepoint_status: 'open', 'restricted', 'closed'
    - time_horizon: months to forecast
    
    Historical baseline (Asia-Europe 40ft container):
    - Peace time: $1,200-2,000
    - 2019 Gulf tensions: $2,200-3,500
    - 2021 COVID/Suez blockage: $8,000-20,000
    - 2024 Red Sea crisis: $3,500-6,000
    """
    
    base_rate = 1800  # USD per 40ft container
    
    # Conflict intensity multiplier
    intensity_multiplier = 1 + (conflict_intensity ** 1.5) * 0.15
    
    # Chokepoint status impact
    chokepoint_factors = {
        'open': 1.0,
        'restricted': 1.4,      # +40% typical during harassment
        'closed': 2.5           # Cape route necessity
    }
    
    # Time decay of disruption (adaptation effect)
    adaptation_factor = max(0.6, 1 - (time_horizon_months * 0.03))
    
    # Insurance premium component (highly volatile)
    if conflict_intensity > 7:
        insurance_component = 800  # War risk premiums
    elif conflict_intensity > 4:
        insurance_component = 400
    else:
        insurance_component = 50
    
    forecast = (base_rate * intensity_multiplier * 
                chokepoint_factors[chokepoint_status] * 
                adaptation_factor + insurance_component)
    
    return {
        'point_estimate': round(forecast),
        'confidence_interval_95': (round(forecast * 0.8), round(forecast * 1.3)),
        'breakdown': {
            'base_freight': round(base_rate * adaptation_factor),
            'conflict_premium': round(base_rate * (intensity_multiplier - 1)),
            'rerouting_cost': round(base_rate * (chokepoint_factors[chokepoint_status] - 1)),
            'insurance_war_risk': insurance_component
        }
    }
```

### 4.2 Oil Price Shock Model

#### Strait of Hormuz Closure Impact

```python
def model_hormuz_closure_impact(closure_duration_days):
    """
    Based on historical supply shocks:
    - 1973 Arab oil embargo: +300% price increase
    - 1979 Iranian revolution: +150% price increase  
    - 1990 Gulf War: +100% price spike (brief)
    - 2022 Russia-Ukraine: +60% (partial supply impact)
    
    Hormuz represents 18-20 million bbl/day
    Global supply buffer: 2-3 million bbl/day SPR releases
    Maximum SPR drawdown: 4.5 million bbl/day (unsustainable beyond 6 months)
    """
    
    baseline_price = 85  # USD/bbl Brent
    hormuz_supply = 18.5  # million bbl/day
    global_supply = 102  # million bbl/day
    spr_drawdown_max = 4.5  # million bbl/day sustainable
    
    # Supply shortfall calculation
    shortfall_pct = hormuz_supply / global_supply  # ~18%
    
    # Price elasticity (short-term inelastic: -0.1 to -0.2)
    # 10% supply drop → 50-100% price increase short term
    price_spike_factor = (shortfall_pct * 5)  # 5x multiplier for inelastic demand
    
    # Duration effects
    if closure_duration_days <= 7:
        panic_premium = 1.3  # Initial shock
        adaptation = 0.0
    elif closure_duration_days <= 30:
        panic_premium = 1.5
        adaptation = 0.1  # Some demand destruction
    elif closure_duration_days <= 90:
        panic_premium = 1.8
        adaptation = 0.25  # Significant demand destruction, SPR releases
    else:
        panic_premium = 2.0
        adaptation = 0.4  # Major economic recession, structural demand drop
    
    price_estimate = baseline_price * (1 + price_spike_factor * panic_premium * (1 - adaptation))
    
    # Economic impact
    if price_estimate > 120:
        recession_probability = min(0.9, (price_estimate - 120) / 100)
        global_gdp_impact = -0.5 - ((price_estimate - 120) / 50)  # Percentage points
    else:
        recession_probability = 0.1
        global_gdp_impact = -0.1
    
    return {
        'oil_price_estimate': round(price_estimate),
        'price_spike_percentage': round((price_estimate / baseline_price - 1) * 100),
        'recession_probability_12m': round(recession_probability, 2),
        'global_gdp_impact_annual': round(global_gdp_impact, 2),
        'key_assumptions': [
            f'Hormuz closure duration: {closure_duration_days} days',
            'No alternative supply routes available',
            'SPR releases at maximum sustainable rate',
            'No peace deal / diplomatic resolution during closure'
        ]
    }
```

### 5.2 Early Warning System Design

#### Multi-Source Intelligence Fusion

```python
class GeopoliticalRiskEarlyWarning:
    """
    Meridian-style early warning system for transportation risk
    """
    
    SIGNAL_SOURCES = {
        'gdelt': {
            'weight': 0.25,
            'indicators': ['conflict events', 'diplomatic tension', 'military exercises'],
            'lag': 'hours'
        },
        'ais': {
            'weight': 0.20,
            'indicators': ['route deviations', 'speed changes', 'clustering patterns'],
            'lag': 'minutes'
        },
        'acled': {
            'weight': 0.20,
            'indicators': ['armed conflict', 'protests', 'violence against civilians'],
            'lag': 'days'
        },
        'insurance_market': {
            'weight': 0.15,
            'indicators': ['war risk premiums', 'underwriter exclusions', 'rate increases'],
            'lag': 'days'
        },
        'satellite': {
            'weight': 0.20,
            'indicators': ['military vessel movements', 'port congestion', 'infrastructure damage'],
            'lag': 'hours'
        }
    }
    
    def calculate_risk_score(self, region, time_window_days=7):
        """
        Composite risk score 0-100 for transportation corridor
        """
        scores = {}
        
        for source, config in self.SIGNAL_SOURCES.items():
            signal_score = self.fetch_signal(source, region, time_window_days)
            scores[source] = signal_score * config['weight']
        
        composite = sum(scores.values())
        
        # Escalation thresholds
        if composite >= 75:
            alert_level = 'CRITICAL'
            action = 'Immediate rerouting recommended'
        elif composite >= 50:
            alert_level = 'HIGH'
            action = 'Contingency plans activated'
        elif composite >= 25:
            alert_level = 'ELEVATED'
            action = 'Enhanced monitoring'
        else:
            alert_level = 'NORMAL'
            action = 'Standard operations'
        
        return {
            'composite_score': round(composite, 1),
            'alert_level': alert_level,
            'recommended_action': action,
            'component_scores': {k: round(v, 1) for k, v in scores.items()},
            'forecast_30d': self.forecast_escalation(region, 30)
        }
```

- **ACLED:** Armed conflict location and event data
- **UN Comtrade:** Bilateral trade flow data
- **World Bank:** Logistics performance index, trade cost data
- **Baltic Exchange:** Freight rate indices (BDI, BCTI, BCCI)
- **Clarksons:** Shipping market intelligence, fleet data
- **IEA:** Oil supply/demand, strategic reserve data
- **FlightRadar24:** Aviation route tracking, airspace closures

### Modeling Methodology
- **Time-Series:** ARIMA/Prophet for freight rate forecasting
- **Monte Carlo:** 10,000+ iterations for disruption scenario modeling
- **Bayesian Networks:** Escalation probability modeling
- **Graph Analysis:** Network vulnerability assessment (chokespoints as nodes)
- **SHAP Explainability:** Feature importance in risk scoring

### Limitations & Assumptions
1. Historical patterns may not predict black swan events
2. Model assumes rational actor behavior (may not hold in ideological conflicts)
3. Climate change impacts on Arctic routes highly uncertain
4. Technological disruption timelines speculative
5. Sanctions evasion undermines official trade statistics
